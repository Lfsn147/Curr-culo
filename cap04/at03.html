2. Ranking de Tecnologias

  1. Angular

    Framework robusto mantido pelo Google, muito usado em aplicações corporativas.

  2. Vue.js

    Framework progressivo e fácil de aprender, ideal para projetos rápidos.

  3. React

    Biblioteca JavaScript popular para construção de interfaces dinâmicas.

  4. CSS

    Responsável pelo estilo e layout das páginas web.

  5. HTML

    Base de toda página web, responsável pela estrutura do conteúdo.

3. Glossário Técnico

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem usada para estilizar elementos HTML.
JavaScript
Linguagem de programação que adiciona interatividade às páginas.
Executada no navegador do usuário.
DOM
Modelo de objetos que representa a estrutura da página HTML.
API
Interface que permite comunicação entre sistemas diferentes.
Pode ser usada para consumir dados de serviços externos.
HTTP
Protocolo HTTP
Protocolo de comunicação utilizado na web.
URL
Endereço Web
Localização de um recurso na internet.
DNS
Sistema que traduz nomes de domínio em endereços IP.
